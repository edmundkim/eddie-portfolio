# Design Preferences

Use this file to capture the current visual direction for the redesign before those choices turn into implementation.

## Current palette

| Token | Hex | Intended use | Notes |
| --- | --- | --- | --- |
| Primary text | `#101010` | Main headings and body copy | Strong contrast without using pure black |
| Accent | `#000BFF` | Links, emphasis, key UI accents | Bold, high-energy blue |
| Secondary text | `#CCCCCC` | Supporting text, captions, quieter metadata | Use carefully so it stays readable |
| Background | `#FBFBFB` | Default page background | Soft off-white that keeps the page bright |

## Quick take

- This palette points toward a minimal, editorial, high-contrast direction.
- The blue accent is vivid enough that it should probably stay rare and intentional.
- `#CCCCCC` may be too light for smaller body text, so it is safest for low-emphasis content rather than anything essential.

## Things to test later

- Heading contrast on `#FBFBFB`
- Link and button states using `#000BFF`
- Whether a darker secondary text color is needed for accessibility in smaller type
