# Reference Assets

Store screenshots, GIFs, short screen recordings, or cropped detail captures here when they help explain a reference.

## Simple guidelines

- Use lowercase, slugged filenames.
- Include the source name and what the asset shows.
- Keep only the assets that are useful for later review.

## Example filenames

- `linear-home-hero.png`
- `linear-scroll-reveal.gif`
- `apple-nav-hover-detail.png`
