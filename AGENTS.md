# AGENTS.md

## Project Goal
Build a modern, minimal portfolio site for Eddie Kim.

The site should communicate:
- Principal Product Designer at AutoScout24
- AI design leadership
- CMO and Co-Founder at CryptoXpress
- Product thinking, systems thinking, and shipped impact

## Style
- Clean
- Minimal
- High quality typography
- Modern but not flashy
- Avoid generic startup website patterns

## Tech Preferences
- Next.js
- TypeScript
- Tailwind CSS
- Simple architecture
- Minimal dependencies
- Mobile-first responsive design

## Rules
- Do not invent fake achievements or metrics
- Prefer small, reviewable changes
- Explain reasoning before major changes
- Keep code readable and modular

## Quality Checks
Before finishing:
- Run lint
- Run typecheck
- Ensure mobile responsiveness
- Check accessibility basics
